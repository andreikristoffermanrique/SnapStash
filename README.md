**Project Description:**

<p align="justify">
SnapStash is a digital photo album application where memories are treated as something as valuable as time itself. Memories are a testament to the present—they influence, inspire, and shape people’s lives. One of the best ways to preserve these moments is through images.<br>
SnapStash is designed to promote accessibility, consistency, and user-friendliness by allowing users to upload, store, and revisit their memories in digital form. The platform enables users to organize photos into albums, making it easier to separate and manage snapshots from different events in a structured and meaningful way.<br>
By addressing the common problem of messy and unorganized digital photos, SnapStash provides a clean and intuitive interface that helps users efficiently manage and relive their captured moments anytime.
</p>

<br>

## How to Run (Windows)

1. **Set up a virtual environment:**
```bash
python -m venv venv
venv\Scripts\activate
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run the application:**
```bash
flask run
```
